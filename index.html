JavaScript is a dynamically typed language where type checking is done during run-time. So, programmers don't have to care about the type of their variables while coding. Previously, JavaScript was meant to be a client-side programming language. Which means that it runs only inside an internet browser, however it's not anymore the case these days, as JavaScript can also run on the server-side using Node.js Javascript runs by v8 engine.

Local Storage

Local storage is used for storing data locally on a user's browser, like saving user preferences or other settings. It is persisted through page reloads and stays with the browser until it is cleared manually or through a setting.

Session Storage

Session storage acts in a similar way to local storage. The only difference here is that while data stored in local storage has an expiration date, the data stored in session storage will expire as soon as the page's session ends, that's mean, when the page is closed.
